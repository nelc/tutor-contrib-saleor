saleor plugin for `Tutor <https://docs.tutor.edly.io>`__
########################################################

Saleor Commerce tutor-plugin to integrate with edx-platform


Installation
************

.. code-block:: bash

    pip install git+https://github.com/nelc/tutor-contrib-saleor

Usage
*****

.. code-block:: bash

    tutor plugins enable saleor
